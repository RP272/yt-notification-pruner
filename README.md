# yt-notification-pruner
Google Chrome extension that removes shorts notifications from YouTube in Web application

## How to use the extension?
1. Download the repository
2. Go to extension page by entering in search bar: `chrome://extensions`
3. Click the `Load unpacked` button
4. Select `yt-notification-pruner` directory
5. The extension should now be visible in the `All Extensions` section
6. Go to YouTube page. The notification section will not include shorts
